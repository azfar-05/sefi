# SEFI
## Software Engineering Failure Intelligence Platform

A PostgreSQL-based system to analyze software development failures using relational queries.

## Tech Stack
- PostgreSQL
- Node.js + Express
- React

## Structure
- /database → SQL scripts (schema, seed, queries)
- /backend → API server
- /frontend → UI

## Goal
Extract insights like:
- Failure-prone files
- Flaky tests
- Regression commits
- MTTR
